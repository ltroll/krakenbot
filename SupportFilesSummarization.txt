Structure of support files.

Local to the bot:

.env file

This file is used to hold secrets and any operational configuration to make bots more flexible.  Any URLs, files, API keys will be stored here.

Example Contents:
KRAKEN_API_KEY=
KRAKEN_API_SECRET=
KRAKEN_API_URL="https://api.kraken.com"
KRAKEN_TICKER_URL="https://api.kraken.com/0/public/Ticker?pair=XXBTZUSD"
KRAKEN_ORDERBOOK_URL="https://api.kraken.com/0/public/Depth?pair=XBTUSD&count=5"
KRAKEN_OHLC_URL="https://api.kraken.com/0/public/OHLC?pair=XBTUSD&interval=60"
LLM_SIGNAL_URL="http://192.168.50.211/bot/llm_signal.json"
BOT_CONFIG_FILE=bot_config.json
BOT_STATE_FILE=last_state.json
BOT_DIR_LIST_FILE=/home/losttroll/tradingbot/bot_dirs.txt
DISCORD_WEB_HOOK=

Bot Configs:

bot_config.json
range_grid_config.json

These are local files specifc to tuning the bot performance.  New bots will likely need their own config with parameters specific to their own purpose.

Supporting Functions

Discord Notifier

discord_notify.py

This is optional and may not be required.  I will mention it if I want it, otherwise don't use it.

It can be invoked with this function: def send_discord_message(message_type="general", message="Insert Test Here"):

pnl_tracker.py

Used for tracking pnl of course.  All transactions are logged to a database.

It is invoked with:

def record_trade(
        self,
        side,
        price,
        volume,
        fee,
        btc_balance,
        usd_balance,
        avg_entry_price,
        current_price
    ):
